# CANADA-PRODUCT-SALES-ANALYSIS
# OVERVIEW
This project analyses product sales from the Canadian retail market dataset. It explores how sales performance varies across product categories, segments, and states while identifying which states and products that drive the highest revenue. 

# DATASET DESCRIPTION
The dataset used for this project was provided by Quantum Analytics. It contained Product ID, Product Name, Category, Segment, Units Sold, Revenue, Manufacturer ID and Name, State, and Date. Key insights derived from the dataset: Revenue by State, Revenue by Product category, Revenue by Segment, Revenue trend over time, Top products by revenue.
